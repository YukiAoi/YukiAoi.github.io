  1. 动画
/* animation：动画名称，一个周期花费时间，运动曲线（默认ease），动画延迟（默认0），播放次数（默认1），是否反向播放动画（默认normal），是否暂停动画（默认running） */
/*执行一次logo2-line动画，运动时间2秒，运动曲线为 linear*/
.a{
  animation: logo2-line 2s linear;
}
